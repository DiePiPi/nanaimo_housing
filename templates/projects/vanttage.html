Project Summary

The project entails building a website using the Django framework in Python. Visitors to the site can query a database for Job titles or Companies of interest to find details pertaining to recruitment processes. This will provide users with information that will help them as they prepare to apply in those companies. The site is primarily geared towards 2 types of users: Employers, and Jobseekers. The database will be populated by visitors to the site. Users can create an account to be able to view their submissions. Users can also submit details without creating an account.

Usability testing will be done with the help of the Graphics Design department at Vancouver Island University. Third year students in the department will test a prototype of the site, and the User Interface will be adjusted based on their feedback. Interviews will be conducted with Employers in Nanaimo as well as Students, to tailor the questionnaires to their expectations. The website owner will be Selena Martin, an Internship Coordinator at Vancouver Island University who has experience guiding others with both sides of the recruitment process.

Project Specs

  • Postgre SQL is used to create a relational database to store data for the website
  • Django/Python is used in a Linux environment, Xubuntu 16.04
  • The web pages are designed using HTML, CSS, and Javascript
  • Once completed the website will be deployed using Heroku
  • The website is being completed under supervision from Dr. Johan Latulippe, a professor at Vancouver Island University.
